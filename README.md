# newproject
a new project skeleton creation utility for multiple languages

  Creates a project directory structure (copies a template from the install location)
  example:
```bash
$ newproject foobar
```
```bash
  ./foobar
  ./foobar/build
  ./foobar/include
  ./foobar/src
  ./foobar/src/main.cpp
  ./foobar/Makefile
```
